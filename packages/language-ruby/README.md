# language-ruby

Ruby support for Lumine.

## Features

- **Grammars**: provides both Tree-sitter and TextMate grammars.
- **Syntax highlighting**: full grammar coverage for Ruby, ERB, Gemfiles, and embedded regular expressions.
- **Snippets**: shortcuts for common Ruby constructs.

## Services

- **hyperlink.injection** (`0.1.0`): consumed to highlight URLs inside Ruby files as clickable links.
- **todo.injection** (`0.1.0`): consumed to highlight `TODO`-style markers inside comments.
