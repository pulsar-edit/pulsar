# autocomplete-html

HTML element and attribute autocompletions in Lumine.

## Features

- **Element completions**: suggests HTML tag names as you type.
- **Attribute completions**: suggests attribute names and values for the current element.
- **MDN descriptions**: annotates suggestions with documentation sourced from MDN.

## Services

- **autocomplete.provider** (`2.0.0`): provided to supply HTML element and attribute suggestions to autocomplete-plus.
