# solarized-light-syntax

A Lumine syntax theme using the popular light Solarized colors.

## Features

- **Solarized palette**: brings the well-known light Solarized color scheme to Lumine.
- **Broad language coverage**: styles tokens consistently across many supported grammars.
- **Bundled by default**: ships with Lumine and is ready to enable from the theme settings.

## Usage

Activate the theme from **Settings > Themes** by selecting "Solarized Light" from the Syntax Themes dropdown.
