# language-html

HTML support for Lumine.

## Features

- **Grammars**: provides both Tree-sitter and TextMate grammars.
- **Syntax highlighting**: full grammar coverage for HTML files.
- **Snippets**: shortcuts for common tags and document scaffolding.
- **Code folding**: collapse elements and comments.
- **Comment toggling**: block comment support.

## Services

- **hyperlink.injection** (`0.1.0`): consumed to highlight URLs inside markup as clickable links.
- **todo.injection** (`0.1.0`): consumed to highlight `TODO`-style markers inside comments.
