# package-generator

Generate and open a new sample package, language, or syntax theme in Lumine.

## Features

- **Package scaffolding**: generate a starter package ready to develop.
- **Language packages**: generate a starter grammar package.
- **Syntax themes**: generate a starter syntax theme.
- **Syntax choice**: create packages in either JavaScript or CoffeeScript.

## Commands

Commands available in `atom-workspace`:

- `package-generator:generate-package`: generate and open a new sample package,
- `package-generator:generate-language-package`: generate and open a new language package,
- `package-generator:generate-syntax-theme`: generate and open a new syntax theme.
