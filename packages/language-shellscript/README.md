# language-shellscript

Shell script support for Lumine.

## Features

- **Grammars**: provides both Tree-sitter and TextMate grammars.
- **Syntax highlighting**: full grammar coverage for Bash, Unix shell scripts, and shell sessions.
- **Snippets**: shortcuts for common shell constructs.

## Services

- **hyperlink.injection** (`0.1.0`): consumed to highlight URLs inside shell scripts as clickable links.
- **todo.injection** (`0.1.0`): consumed to highlight `TODO`-style markers inside comments.
