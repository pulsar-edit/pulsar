# update-package-dependencies

Install the dependencies of the current Lumine package project.

## Features

- **One-command install**: installs every dependency from the project's `package.json` into `node_modules`.
- **Progress indicator**: shows install progress in the status bar while running.
- **Package projects**: intended for use in projects that are Lumine packages.

## Commands

Commands available in `atom-workspace`:

- `update-package-dependencies:update`: install the dependencies of the current project.

## Services

- **status-bar** (`^1.1.0`): consumed to show an install progress indicator in the status bar.
